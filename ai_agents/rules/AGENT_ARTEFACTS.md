# Agent Artefacts

## system_lead_agent

- roadmapa kroku
- task breakdown
- decyzja delegacyjna
- status realizacji

## architecture_agent

- propozycja zmiany architektury
- impact note
- update dokumentacji technicznej

## control_layer_agent

- spec modułu `core/`
- plan implementacji
- checklista testowa

## strategy_agent

- hipoteza strategii
- raport backtestu
- rekomendacja: odrzuć / popraw / promuj dalej

## api_agent

- kontrakt API
- opis endpointów
- update OpenAPI

## gui_agent

- spec ekranów
- flow użytkownika
- mapowanie GUI do API

## monitoring_agent

- lista metryk
- plan dashboardów
- zasady feedback loop

## integration_agent

- plan integracji
- lista zależności
- checklista kompatybilności

## review_agent

- raport review
- klasyfikacja ryzyka
- lista blockerów
- decyzja: approve / revise / escalate
