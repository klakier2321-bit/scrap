# Rules

Ten katalog będzie zawierał reguły pracy agentów.
Reguły mają ograniczać chaos, zmniejszać ryzyko i pilnować zgodności działań agentów z zasadami projektu.

Znajdują się tu między innymi:
- zasady granic odpowiedzialności agentów
- macierz ownership agentów
- zasady zmian cross-layer
- manifest scope w formacie czytelnym dla przyszłego runtime agentów

Na tym etapie znajdują się tu:
- zakres decyzyjny agentów
- zasady eskalacji
- lista oczekiwanych artefaktów agentów
- macierz ownership agentów
- zasady zmian cross-layer
