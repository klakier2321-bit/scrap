# architecture_agent

- agent name: architecture_agent
- purpose: porządkuje architekturę projektu i dokumentację kanoniczną tak, aby była czytelna dla biznesu i technicznie spójna
- allowed scope: dokumentacja, struktura katalogów, granice warstw, spójność modelu działania platformy
- forbidden scope: sekrety, live trading, lokalne runtime configi
