# Monitoring Reports

Ten katalog jest jedynym dozwolonym miejscem zapisu dla artefaktów `control_status`.

- `control_status_SUMMARY.md`
  Krótkie podsumowanie po polsku dla prezesa i operatora.
- `control_status.json`
  Zanonimizowany, ustrukturyzowany widok do dalszej agregacji.

Zasady:
- brak surowych ścieżek systemowych,
- brak sekretów i identyfikatorów,
- brak zapisu poza `monitoring/reports/`.
